# Manbhari - Pure Organic Spices & Snacks

A premium e-commerce platform for authentic Indian organic masalas, homemade snacks, and puja items.

## Features
- **Organic Masala**: Pure, traditionally crafted spices.
- **Homemade Snacks**: Healthy and tasty traditional Indian snacks.
- **Puja Items**: Spiritual essentials.
- **Admin Dashboard**: Manage products, images, and orders.
- **Wallet System**: Referral-based wallet for discounts.
- **WhatsApp Integration**: Direct order confirmation via WhatsApp.

## Tech Stack
- React + Vite
- Tailwind CSS
- Firebase (Firestore & Auth)
- Framer Motion (Animations)
- Lucide React (Icons)

## Deployment
This project is configured for automatic deployment to **GitHub Pages** via GitHub Actions.
Every time you sync from AI Studio, the site is rebuilt and updated.
